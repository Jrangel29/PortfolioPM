Find the voice

Mini-Jogo feito na cadeira de laboratório 3

Um mini-jogo desenvolvido em grupo com o intuito de perceber desenvolver as nossas capacidades na linguagem JavaScript.

Este projeto dividiu-se em 2 partes principais. A primeira parte consistia num jogo 2D inspirado no "Super Mario", no qual o jogador tem que desviar obstáculos e apanhar itens. A segunda parte consistia num jogo da memória.

A Somar

Aplicação mobile feita na cadeira de laboratório 4

Este projeto foi feito em conjunto com a BioLiving, com o objetivo de se promover atitudes mais sustentáveis e amigas da natureza. Baseou-se especialmente em PHP e MySQL, permitindo-nos perceber mais sobre gestão de bases de dados.

A aplicação permitia aos utilizadores publicar/receber boleias para eventos da BioLiving, podendo, depois do evento, avaliar a boleia. Para além disso, os utilizadores podiam ganhar pontos nos eventos, o que os permitia ganhar subir no ranking da aplicação e trocar os pontos por prémios na aplicação.

F.S.S.U.P.

Base de dados de super-heróis feita na cadeira de laboratório 5

Este projeto foi desenvolvido com o intuito de desenvolver as nossas capacidades de React e perceber como funcionam e podem ser geridos os pedidos API.

O projeto consiste numa base de dados de super-heróis Marvel e DC. Estes são separados de acordo com o seu universo, nos quais o utilizador pode ser uma lista de todos os super-heróis de cada editora, podendo depois aceder a detalhes de cada um.
